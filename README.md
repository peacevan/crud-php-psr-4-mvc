# Como executar o projeto

## Back end
Pré-requisitos: PHP 7, PDO MYSQL

```bash
# clonar repositório
git clone https://github.com/peacvan/crud-php-psr-4-mvc

# entrar na pasta do projeto back end
cd backend

# executar o projeto
hp -S localhost:8000

```
